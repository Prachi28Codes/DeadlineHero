# 🚀 Deadline Hero

Deadline Hero is an AI-powered productivity web app that helps students manage study plans, track tasks, and stay consistent using smart planning tools.

---

## ✨ Features

- 🤖 AI Study Plan Generator (Gemini API)
- 📅 Daily Task Manager
- 📊 Progress Tracker
- 🎤 Voice Input Support
- 🌙 Dark / Light Mode
- 🧠 Smart Priority System
- 📥 Copy & Download Plan

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Google Gemini API

  AI integration powered by Google Gemini
  API (Google AI ecosystem)
